## 🎯 Business Objective
This automation downloads daily "Pending Items" extract files generated by legacy ERP mainframes, safely parses massive (potentially unformatted or disguised HTML) files, intelligently filters the data based on complex business metrics (`Returnable="Y"`), and safely maps the results into a dynamic Master Dashboard Excel Template cleanly.

## 🚀 Key Technical Features
* **Hybrid Execution Framework:** Orchestrated via `.robot` for modular enterprise task execution while leveraging Python Pandas internally for high-speed, data-frame manipulations before ever touching Excel COM operations.
* **Auto-Resilient Ingestion Pipeline:** Uses a custom multi-format ingestion engine `(pyxlsb -> openpyxl -> xlrd -> memory-efficient manual HTML parser)`. Even if the source system outputs a disguised `.xls` (HTML in reality), the module gracefully catches pandas `MemoryError` issues and parses it manually.
* **Template Drift Protection:** Instead of statically pasting data to `A2`, the script dynamically scans the vertical Dashboard column layout to locate where true data rows start, perfectly adjusting to standard human modifications in the template structure.
* **Granular Formula Injection:** Calculates precise structural alignment natively via DataFrame dimensions out to `.formula`, automatically bridging raw exported data to complex Excel KPI logic.
* **Zero-Hanging Processes:** Strictly manages COM execution `App.Quit()` with explicit teardowns (`try-except-finally`), guaranteeing zero rogue `Excel.exe` instances are orphaned on Host Machines post-execution.

## 📁 Files & Structure
* `pending_report.py`: Core processing engine housing all Pandas data cleaning, alignments, and XLWings formula injections.
* `pending_report.robot`: Standardized Robot Framework wrapper providing transparent external parameters, native dashboarding logic, and reliable execution metrics.

> **Note for Reviewers:** Due to Data Privacy guidelines, exact domain logic formulas inside the Excel integrations and paths to actual raw `.xlsb` samples have been masked or generalized within this repository.
